# Design Proposal Archive

`/.proposals/.archive/design/` stores archived manifest-governed design
proposals.

Archived packages remain temporary, non-canonical implementation material. They
stay here only as historical reference after implementation or historical
retention.

Rules:

- every archived proposal must live at `/.proposals/.archive/design/<proposal-id>/`
- every archived proposal must include `proposal.yml`
- design proposals should also include `design-proposal.yml`
- proposal manifests remain the lifecycle authority
- live implementation targets must not depend on archived proposal paths
