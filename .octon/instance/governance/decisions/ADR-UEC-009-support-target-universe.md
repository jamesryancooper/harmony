# ADR-UEC-009 Support Target Universe

- Status: active
- Date: 2026-04-08
- Decision: The full target support universe includes both model tiers, all
  four host adapters, all three workload tiers, both language-resource tiers,
  both locale tiers, and all six capability packs.
- Scope: support-target policy and release claims
- Canonical refs:
  - `/.octon/instance/governance/support-targets.yml`
  - `/.octon/framework/engine/runtime/adapters/{model,host}/**`
- Reopen only if Octon intentionally changes its target support universe.
