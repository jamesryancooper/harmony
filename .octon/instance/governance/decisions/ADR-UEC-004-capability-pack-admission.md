# ADR-UEC-004 Capability Pack Admission

- Status: active
- Date: 2026-04-08
- Decision: Capability packs are first-class governed artifacts under
  `/.octon/instance/governance/capability-packs/**`, with explicit route,
  admission, and disclosure semantics.
- Scope: capability-pack governance and admission
- Canonical refs:
  - `/.octon/framework/constitution/contracts/adapters/capability-pack-v1.schema.json`
  - `/.octon/instance/governance/capability-packs/registry.yml`
  - `/.octon/instance/governance/exclusions/action-classes.yml`
- Reopen only if new action surfaces or risk classes are introduced.
