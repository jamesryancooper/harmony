# Create Context Doc Minimal

## Target Kind

`context-doc`

## Preconditions

- `demo-pack` already exists

## Invocation

- `/octon-pack-scaffolder-create-context-doc --pack-id demo-pack --doc-id usage-notes`

## Expected Outputs

- `context/usage-notes.md`
- descriptive, additive-only content

## Forbidden Outputs

- governance contracts
- direct runtime authority
