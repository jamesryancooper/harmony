# Report Mission Creation

Summarize the completed mission creation outcome.

## Report Must Include

1. The mission authority root.
2. The mission control root.
3. The mission continuity root.
4. The generated route and mission-view paths.
5. The initial paused/healthy posture.
6. The next required operator action before material autonomous work begins.
