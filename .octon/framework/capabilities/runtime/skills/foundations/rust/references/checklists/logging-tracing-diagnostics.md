# Checklist: Logging, Tracing, Diagnostics

- [ ] Verbosity flag exists when diagnostics matter
- [ ] Quiet mode exists when automation/CI needs clean output
- [ ] Structured events exist for long-running/consequential tasks
- [ ] `tracing` spans wrap meaningful work
- [ ] Progress reporting exists for long operations
- [ ] Dry-run exists for mutations
- [ ] Explain mode exists for complex decisions
- [ ] Final summary/audit output exists
- [ ] Logs do not leak secrets
