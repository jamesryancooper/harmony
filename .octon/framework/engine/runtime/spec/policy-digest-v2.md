# Octon Policy Digest v2

Policy digests emitted for Mission-Scoped Reversible Autonomy runs must include,
at minimum:

- run id
- policy decision
- effective ACP level
- mission id
- slice id
- oversight mode
- execution posture
- reversibility class
- rollback handle or compensation handle
- recovery window
- autonomy budget state
- breaker state
- reason codes
- remediation summary

Digests remain derived summaries over canonical receipts and must not become a
second authoritative control ledger.
