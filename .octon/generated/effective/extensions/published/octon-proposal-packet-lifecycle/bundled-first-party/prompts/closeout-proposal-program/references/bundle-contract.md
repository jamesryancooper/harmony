# Bundle Contract

The parent closeout route refuses archival while any required child is planned,
created, validated, ready, implementing, correction-needed, or blocked without
an explicit deferral, supersession, or rejection receipt.

Closeout writes only parent-local `support/proposal-closeout.md`; it may
summarize child outcomes but never satisfies child receipts, child promotion
targets, child validation verdicts, child archive metadata, or child terminal
outcomes.
