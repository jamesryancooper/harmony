# Step 3: Select Workflow Template

## Purpose

Choose the canonical workflow scaffold, stage layout, and generated README shape
 before creating files.

## Actions

1. Use the canonical scaffold:
   `.harmony/orchestration/runtime/workflows/_scaffold/template/`
2. Plan the canonical files:
   - `workflow.yml`
   - `stages/`
   - `README.md`
   - `guide/NN-*.md`
3. Decide whether the workflow also needs optional:
   - `schemas/`
   - `fixtures/`
   - `_ops/`
   - `references/`

## Output

- Selected workflow scaffold
- Stage asset list
- Guide generation plan

## Proceed When

- [ ] Canonical workflow scaffold exists
- [ ] Stage asset list is complete
- [ ] Guide layout is fixed
