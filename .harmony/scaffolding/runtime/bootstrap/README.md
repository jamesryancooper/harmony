# Bootstrap Assets

Canonical repo-bootstrap assets used by `/init`.

## Contents

- `init-project.sh` - canonical bootstrap implementation
- `AGENTS.md` - canonical authored AGENTS source rendered to `/.harmony/AGENTS.md`
- `BOOT.md` - optional recurring startup checklist template
- `BOOTSTRAP.md` - optional one-time bootstrap checklist template
- `alignment-check` - root shim template
- `objectives/` - objective brief and intent-contract starter packs rendered to `/.harmony/OBJECTIVE.md` and `/.harmony/cognition/runtime/context/intent.contract.yml`
- `manifest.yml` - bootstrap bundle metadata for projection and validation

Repo-root `AGENTS.md` and `CLAUDE.md` are ingress adapters to `/.harmony/AGENTS.md`.
